# Блок-схема проекта

```mermaid
flowchart TD
    A[Начало: Загрузка текстового файла text.txt<br>• Обязательно >=100 строк текста<br>• Гуманитарная тематика: литература, песни, статьи] --> B[Вывод количества строк в файле на экран<br>• Использование len(file.readlines())<br>• Комментарий: "Исходный текст: [название произведения]"]
    B --> C[Базовая обработка текста:<br>• Разделение на предложения: . ! ?<br>• Разделение на слова: split(), очистка от пунктуации<br>• Приведение к нижнему регистру]
    C --> D{Применение структур данных<br>• Списки: хранение слов и предложений<br>• Множества: уникальные слова<br>• Словари: частоты слов, буквы-счетчики}
    D --> E[Создание пользовательской функции:<br>• Например: count_word_frequency(word)<br>• Возвращает количество вхождений слова<br>• Вызывается для пользовательского запроса]
    E --> F[Подсчет >=5 текстовых характеристик:<br>1. Общее количество слов<br>2. Общее количество предложений<br>3. Средняя длина слова (символов)<br>4. Средняя длина предложения (слов)<br>5. Топ-10 частотных слов (без стоп-слов)<br>6. Количество уникальных слов<br>7. Частота заданного пользователем слова<br>8. Распределение слов по первой букве алфавита]
    F --> G[Использование циклов (for/while) корректно:<br>• for word in words: для подсчета частот<br>• for sentence in sentences: для длины предложений<br>• while для обхода символов при фильтрации]
    G --> H[Применение условных конструкций (if/elif/else):<br>• Исключение стоп-слов: if word in stop_words: continue<br>• Фильтрация пустых строк: if not word: skip<br>• Проверка наличия слова в словаре: if word in freq: ... else: ...]
    H --> I[Вывод результатов на экран:<br>• Четкая структура: "Количество слов: X"<br>• Топ-10 слов: построчный вывод<br>• Распределение по буквам: "A: 15, B: 8, ..."]
    I --> J[Сохранение результатов в results.txt:<br>• Открывается в режиме 'w'<br>• Все выводимые данные записываются в файл<br>• Файл закрывается корректно]
    J --> K[Оформление кода в .ipynb:<br>• Ячейки: загрузка → обработка → вывод → сохранение<br>• Каждый этап — отдельная ячейка<br>• Комментарии >=5 строк: объясняют логику, функции, выбор стоп-слов]
    K --> L[Финал: Сдача проекта<br>• Без внешних библиотек (только стандартные: re, string, io)<br>• Нет избыточного кода<br>• Полная гуманитарная тематика<br>• Все критерии выполнены]
